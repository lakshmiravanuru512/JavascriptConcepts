# Map, Filter And Reduce #

```
Map(): It takes an array and it returns the array the length of both input array and the output array must be same. it will take a call back function and returns an array.

Filter(); It takes input as an array. It will filter the given array based on the given condistion and it returns an array.

Reduce();  It takes input as an array. It will take a call back function and it takes two arguments as accumulator and the currentvalue. And it reuces to single value based on the given consition.
```
